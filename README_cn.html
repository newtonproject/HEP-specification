# DApp类型

HEP支持三种类型的Dapps:

Dapp类型 描述 开发语言
DMA - 去中心移动应用 在iOS或安卓上运行。可在苹果商店和google商店上下载 Swift(iOS)或者Java(Android)
DWEB - 去中心网站 接入HEP api的标准网站 Javascript
NewDApp 在NewVM上运行对去中心应用. 与微信小程序相似 Javascript或者 NewLang
